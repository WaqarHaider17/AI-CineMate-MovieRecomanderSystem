# CinéMate Intelligent Movie Recommender

## Overview
CinéMate is an intelligent movie recommendation application built using Python's tkinter library for the GUI, pandas for data manipulation, scikit-learn for text vectorization and cosine similarity computation, and difflib for fuzzy matching. It provides movie recommendations based on the similarity of selected features including genres, keywords, tagline, cast, and director.

## Features
- Allows users to input their favorite movie.
- Recommends similar movies based on selected features.
- Provides links to watch recommended movies if available.
- User-friendly GUI design using themed widgets.

## Installation
1. Clone or download the repository.
2. Install the required dependencies:
3. Make sure you have Python installed on your system.
4. Run the `main.py` file to start the application.

## Usage
1. Launch the application by running `main.py`.
2. Enter the name of your favorite movie in the input field.
3. Click on the "Get Recommendations" button.
4. The application will display a list of recommended movies along with buttons to watch them (if available).
5. Click on the "Watch" button to open the movie's homepage link in a web browser.

## Notes
- Make sure the `movies.csv` file is present in the same directory as the `main.py` file.
